🌐 Chenn的个人主页介绍

我的朋友,欢迎来到我的个人网站！这是一个承载兴趣、学习、游戏与灵感的数字空间。

🏠 网站包含以下模块

  • 主页展示（Home） —— 使用 TypeIt 和动态特效打造的界面(文案来自《崩坏：星穹铁道》)。
  • Projects —— 展示我进行的项目、学习成果和分享内容。
  • 音乐播放 —— 在头像以及设置处可以播放音乐,使用 MetingJS + Aplayer 接入网易云歌单。
  • 访客交互 —— 包括打赏页和神秘链接,未来将加入更多功能(画饼，有时间就做~)。

📘 技术栈

  • Vue3 + Vite
  • Cloudflare Pages 部署
  • Aplayer + MetingJS 音乐播放器
  • 自定义 CSS + 动态背景 + 卡片交互效果
  • 最主要的: github大佬开源(红豆泥阿里嘎多!)

✨ 关于这个网站

这个网站从视觉到交互都进行了高度定制，目标是打造一个 “轻量、美观、有趣的数字名片”。 我会在这里分享我的学习笔记、项目成果、游戏测试经验、AI 研究内容等等。
